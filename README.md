# WhatsApp Bot - WPPConnect

Bot de atendimento automático para WhatsApp utilizando WPPConnect.

## 🚀 Funcionalidades

- Saudação automática (Bom dia / Boa tarde / Boa noite)
- Resposta fora do horário de funcionamento
- Envio automático de cardápio
- Janela de 24h respeitada
- Envio de imagens do cardápio
- Resposta automática para qualquer mensagem recebida

---

## 🛠 Tecnologias Utilizadas

- Node.js
- @wppconnect-team/wppconnect

---

## 📦 Requisitos

- Node.js 20+ (LTS recomendado)
- 4GB RAM mínimo
- Internet estável
- Windows 10/11 ou Linux
